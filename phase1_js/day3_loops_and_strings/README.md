# Day 3 — Loops and Strings

Small examples and exercises demonstrating JavaScript loops, string operations, and a simple CLI number-guessing game.

## Files
- [loopsAndStrings.js](loopsAndStrings.js) — Examples of `for`, `while`, `do-while`, `for...of`, and `for...in` loops, plus string manipulation and template literals (see [`greeting`](loopsAndStrings.js)).
- [numberGuessingGame.js](numberGuessingGame.js) — CLI number guessing game using `prompt-sync`; main settings are [`targetNumber`](numberGuessingGame.js) and [`maxAttempts`](numberGuessingGame.js).
- [package.json](package.json) — dependency list (uses `prompt-sync`).

## Prerequisites
- Node.js (v12+ recommended)
- npm
